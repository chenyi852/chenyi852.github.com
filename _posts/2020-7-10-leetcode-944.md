思路：
贪心算法
1 按列编列字符串数组， 一直到字符串终止
2 每次循环，判断上一个字符是否比当前字符大，如果前一个字符大，则说明是降序，退出此列比较，同时将ans加1，说明此列可以删除。


```
int minDeletionSize(char ** A, int ASize){
    int ans = 0;
    int col = -1;

    if (ASize == 1) {
        return 0;
    }

    while (A[0][++col] != 0) {
        for (int row = 1; row < ASize; row++) {
            /* non-desending order */
            if (A[row - 1][col]  > A[row][col]) {
                ans++;
                break;
            }
   
        }
        
    }
    return ans;
}
```
